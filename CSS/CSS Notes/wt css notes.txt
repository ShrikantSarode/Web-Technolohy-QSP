*CSS bg-image:-

The bg-img property specify's an image to used the bg of an element.
By default, the image is repeated so it's covered the entire element.

example-

body{
	bg-img:url(./);
	}

*Css bg-repeate: no-repeat-
Showing the bg-img only once is also specify's by the bg-img property.
ex:-

body{
	bg-img:url(./);
	bg-repeat:no-repeat;	
}

*CSS BG-POSITION :-

the css bg property is used to specify the position of bg-img.
ex:-

body{
	bg-img:url(./);
	bg-repeat: no-repeat
	BG-position: top-right;
	}

*CSS BG-ATTACHMENT:-
The bg-attachment properties specifies wether the bg-img should scroll or be fixed.(Will not scroll with the rest of the page).
EX-
1-Specify that the bg-img should be fixed.
body{
	bg-img:url(./);
	bg-repeat: no-repeat
	BG-position: right-top;
	bg-attachment: fixed;
	}
2-Specify that the bg-img should be scroll.
body{
	bg-img:url(./);
	bg-repeat: no-repeat
	BG-position: right-top;
	bg-attachment: scroll;
	}

css bg-size:-
the bg-size properties specify's the size of bg-img.
ex-
body{
	bg-img:url(./);
	bg-repeat: no-repeat;
	bg-size: cover;
	}

*CSS OVERFLOW-

the overflow properties specify's welther to clip the content or to add scroll bars when the content of the element is too big to fit in specificed area.
The overflow properties has following values:-
1-visible(default)
2-hidden
3-scroll

1-Visible-
The overflow is not clipped.The content render outide the box.

2-Hidden
The overflow is clipped and rest of the content is invisible.

3-Scroll-
The overflow is clipped, and a scroll bar is added to see the rest of the content.

4-Auto -

Auto is simillar to scroll but it add only when the content is overflow.

*OVERFLOW X AND OVERFLOW Y-
the overflow-x,y properties specify wether to change the content is just vertical or horizontaly or both-

1-Overflow x specify's what to do with the left-right edges of the content.

2-overflow-y specify what to do with top-bottom content of edges of y.

Flex-direction Property:-The flex direction property defines in which direction the container wants to flow the flex items.

.parent{
display:flex;
flex-direction:row;(default)
}

#child1{
	h-100px;
	w-100px;
}
flex-flow:- It is a shortend properties for the setting both the flex-flow property.

case1:
.parent{
display:flex;
flex-flow: row wrap;
}
case2:
.parent{
display:flex;
flex-flow: row-reverse wrap;
}

align-content Property:-The align-content Property is used to align the flex lines. (rows/columns)

.parent{
display:flex;
align-content:space-between;
flex-flow: row wrap;
}

.parent{
display:flex;
align-content:space-between;
flex-flow: column wrap;
}

Note:- The align-item property is changing the alignment according to it's direction.

1-If flex direction is row then, align content property align the lines about cross axis.

2-If flex direction is column then, align content property align the lines about main axis.

The Order Property:-The Order Property Specifies the order od the flex items.

.parent{
display:flex;
order:1;
}
.child{
text-align:center;
}

Deafult order of the child boxes is zero.
.parent{
display:flex;
}
#child{
order:4;
}

Note: The order value must be a number, default value is zero.

The flex-grow Property:-The flex-grow Property specifies how much a flex item will grow relative to the rest of the flex item.

.parent{
display:flex;
}
#child3{
flex-flow:5;
}
Note: The order value must be a number, default value is zero.

The flex-shrink Property:-The flex-shrink Property specifies how much a flex item will shrink relative to the rest of the flex item.
.parent{
display:flex;
}
#child2{
flex-shrink:0;
}

Note: default values is 1.

The flex-basis Property:-The flex-basis Property specifies the initial length of flex item.

#child2{
flex-basis:100px
width:50px;
}

The flex Property:-The flex Property is a shortend property for the flex-grow, flex-shrink,and flex-basis properties.

.parent{
flex: 0px 0px 0px;
}
